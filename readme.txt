== Description ==

The custom Lana Widgets are optimized for Bootstrap.

= Widgets list =
* Lana Alert
* Lana Carousel
* Lana Image
* Lana Jumbotron
* Lana Page Content
* Lana Panel
* Lana Text
* Lana Thumbnail
* Lana Title
* Lana Well


= Lana Blog =
[Lana Widgets](http://wp.lanaprojekt.hu/blog/wordpress-plugins/lana-widgets/)

== Installation ==

= Requires =
* WordPress at least 4.0
* PHP at least 5.3

= Instalation steps =

1. Upload the plugin files to the `/wp-content/plugins/lana-widgets` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
